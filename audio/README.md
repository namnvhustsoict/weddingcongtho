## Audio Specifications

### Recommended:
- **Format**: MP3 (best browser compatibility)
- **Bitrate**: 128kbps to 192kbps
- **File size**: Under 5MB
- **Duration**: 2-4 minutes (will loop automatically)

### Alternative formats supported:
- `.mp3` - Best compatibility
- `.ogg` - Good for Firefox
- `.wav` - High quality but larger file size
